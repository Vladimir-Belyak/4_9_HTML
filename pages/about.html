Коротко о возможностях HTML

Язык разметки HTML позволяет создавать простые сайты-визитки. Как правило наполненные текстовой информацией, которую например, можно выравнивать по ширине(этот текст)

по левому краю,

по центру,

по правому краю.

      • Йогурт
